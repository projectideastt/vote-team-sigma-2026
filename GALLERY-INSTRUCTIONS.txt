Team Sigma Gallery Instructions

This package now uses an image-only gallery for election-day browsing.

Key files:
- gallery/index.html: gallery page
- gallery.js: reads the JSON manifest and renders filters, image grid and lightbox
- assets/gallery/gallery-manifest.json: list of images used by GitHub Pages
- scripts/generate-gallery-manifest.js: regenerates the JSON file after images are added

Folder structure:
- assets/gallery/images/ -> campaign images

To add more images:
1. Copy images into assets/gallery/images/.
2. Run this from the repository root:

   node scripts/generate-gallery-manifest.js

3. Commit and push the updated image files and assets/gallery/gallery-manifest.json to GitHub.

GitHub Pages cannot automatically scan folders, so the JSON manifest is the clean, reliable way for the static gallery page to know which images to display.
